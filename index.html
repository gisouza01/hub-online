Seja bem-vindos ao meu projeto!

Olá! Sou Giovanna Souza e este é um projeto desenvolvido para a matéria de Desenvolvimento de Interfaces. Estou no início do curso de Análise e Desenvolvimento de Sistemas e cada página aqui reflete o meu aprendizado e a minha paixão por tecnologia. Explore as seções para ver o que tenho construído!

Linguagens e Tecnologias que estou Aprendendo

HTML

A base de toda a web! Com HTML, estou aprendendo a estruturar o conteúdo das páginas, desde textos e imagens até formulários complexos, garantindo que a informação seja organizada e acessível.

CSS

Responsável por dar vida e estilo às páginas. Com CSS, estou explorando como tornar o design visualmente atraente, responsivo e interativo, aplicando cores, fontes, layouts e animações para uma experiência de usuário envolvente.

JavaScript

A linguagem que traz interatividade e dinamismo para a web. Estou começando a entender como manipular elementos HTML, responder a eventos do usuário e criar funcionalidades que tornam as páginas mais ricas e funcionais.
